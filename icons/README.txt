ICONS REQUIRED

This folder must contain three icon files for the Chrome extension:

1. icon16.png  (16x16 pixels)
2. icon48.png  (48x48 pixels)
3. icon128.png (128x128 pixels)

HOW TO CREATE ICONS:

Method 1 (Recommended):
- Open the create-icons.html file in your browser
- Icons will be generated automatically
- Click download buttons to save them here

Method 2 (Custom Icons):
- Use any image editor (Photoshop, GIMP, etc.)
- Create three PNG files at the sizes listed above
- Use YouTube or comment-related imagery
- Save them in this folder with the exact names above

Method 3 (Online Tools):
- Visit https://favicon.io/ or similar services
- Upload your design or use text/emoji
- Generate all sizes
- Download and rename to match the required names

IMPORTANT:
Without these icons, the extension will not load properly in Chrome.
